Equity research agent

Delayed market data (yfinance) plus Google News RSS headlines, orchestrated with LangGraph (fetch → validate → draft → review). The LLM runs on MiniMax via the OpenAI-compatible API (default model MiniMax-M2.5). Not investment advice.

Open-source lineage. This demo is not a fork of the repos below; it is our own stack. The professional reference on GitHub for multi-agent finance is TauricResearch/TradingAgents (multi-agent LLM trading / analysis framework, high star count). Orchestration here uses langchain-ai/langgraph. Data: public delayed quotes and RSS aggregation only.

LangGraph: fetch → validate → draft → review (ComfyUI-style node layout)
