股票分析师智能体

使用 yfinance 延迟公开行情与 Google News RSS 标题聚合，经 LangGraph 四步编排： 拉数 → 校验 → 写稿 → 复核。大模型通过 MiniMax 官方 OpenAI 兼容接口 调用（默认模型 MiniMax-M2.5，与您会员线路一致请在环境变量中核对）。 不构成投资建议。

开源参考与说明。 本演示并非下列仓库的 fork，为本站自研组合实现。职业向多智能体金融开源参考推荐 GitHub TauricResearch/TradingAgents（多智能体 LLM 金融交易/分析框架，社区星标较高）。 编排采用 langchain-ai/langgraph。 数据仅为公开延迟行情与 RSS，不含付费终端级实时数据。
